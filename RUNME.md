# RUNME.md — 铁冠计划 MVP 演示

## 环境
- Unity 6000.0.76f1（已安装）

## 步骤
1. 用 Unity Hub 打开 `E:\IronCrown` 项目
2. 菜单 **IronCrown > Setup Main Scene**（首次需要，创建 `Assets/Scenes/Main.unity`）
3. 打开 `Assets/Scenes/Main.unity`
4. 点击 **Play**

## 预期
- HUD 显示：回合 1 · TurnStart
- 6 个国家及各自国库、稳定度、资源
- 点击「推进」按钮 → 回合/阶段前进 → 数字逐回合变化

## 验证
- 推进 1 回合后，有资源产出的国家资源应增加
- 装备库存随军工厂数量变化
