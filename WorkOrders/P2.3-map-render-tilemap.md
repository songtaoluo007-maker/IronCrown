# P2.3 — 地图渲染换栈（世界空间 Tilemap + 相机）

## 地位 / 依赖
**前置:P2.2 通过**(需 TileState/tileIds 已在数据层)。对应 `Design/MAP_ARCHITECTURE.md` §5 + §8 第 2 步(渲染层)。**表现层大改,呈现层历来坑多** — 严格按 DoD 出 PlayMode + 真实截图。

## 范围（照 MAP_ARCHITECTURE §5）
把地图从 **UI Toolkit 绝对定位色块** 换成 **世界空间 Tilemap + 正交相机**;UI Toolkit **退回只做 HUD/面板浮层**。
1. **Tilemap 渲染**:用 `com.unity.2d.tilemap`(已在 manifest)。按 tile.gridX/gridY 摆格,按 controllerCountry 上色(省级配色),格地形先用占位色块/简单 tile(贴图美化留 D)。
2. **正交相机**:`SetupScene` 建正交相机渲染 Tilemap;支持缩放/平移(基础)。
3. **选省命中**:点击 → 相机射线/`Tilemap.WorldToCell` → 命中 tile → 映射 `provinceId` → 复用现有 `GameSessionService.SelectProvince`。
4. **拆旧**:移除 UI Toolkit 里的色块地图渲染(`MapView`/绝对定位省块),HUD 其余面板保留。
