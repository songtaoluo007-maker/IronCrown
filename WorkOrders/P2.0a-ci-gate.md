## ★ 需要你（人类）做的一步：配 Unity license（启用 test job）
lint 已生效。Unity test job 需 license（game-ci,Personal 免费）：
1. **生成 .alf**：在 Actions 跑一次 `game-ci/request-activation-file`（或本地 `Unity -batchmode -createManualActivationFile`）→ 拿到 `Unity_v6000.x.alf`。
2. **换 .ulf**：浏览器开 https://license.unity3d.com/manual → 上传 .alf → 下载 `.ulf` 许可文件。
3. **复制 .ulf 全文**（XML）。
4. **配 Secret**：GitHub 仓库 → Settings → Secrets and variables → Actions → New repository secret：
   - Name = `UNITY_LICENSE`
   - Value = 粘贴 .ulf 全文
5. 下次 push,test job 自动启用,跑 EditMode + PlayMode + 上传结果 artifact。

> 备选(Plus/Pro)：用 `UNITY_SERIAL`+`UNITY_EMAIL`+`UNITY_PASSWORD` 三个 secret,并把 workflow `env` 改成对应字段。
